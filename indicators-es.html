Indicadores de enfermedad
cardíaca en Excel

Este artículo explica cómo se llevó a cabo el proyecto
y habla acerca de las herramientas utilizadas.

¿De qué se trata el proyecto?

En este proyecto utilicé distintos métodos y herramientas para analizar y limpiar la información de una base datos relacionada con un estudio acerca de enfermedades cardíacas. Posteriormente, elaboré y diseñé un "dashboard" para mostrar los resultados y la información más importante obtenida a partir del estudio.

¿De dónde se obtuvieron los datos?

Toda la información se obtuvo a partir de una base de datos encontrada en el conocido sitio web de análisis de datos "Kaggle". Este set de datos muestra la información de diversos estudios realizados por 5 centros médicos y hospitales, entre los que destacan el Hungarian Institute of Cardiology y el V.A. Medical Center, Long Beach and Cleveland Clinic Foundation.



Cómo se llevó a cabo el proyecto

Los datos

Lo primero fue elegir una base de datos que pudiera funcionar para el proyecto. Quería utilizar un set de datos que, a pesar de contar con información real y precisa, tuviera algunos errores pequeños o formatos distintos que yo pudiera corregir o modificar. Decidí utilizar la base de datos de este estudio cardiológico, pues cumplía con esos requisitos. De igual manera, se trata de un tema que me parece interesante y de un estudio que permite obtener diversas conclusiones.

Preparación

Antes de iniciar me gusta preparar algunas cosas para facilitar el trabajo posterior. La primera es elegir con qué formato de archivos trabajaré. En este caso, elegí convertir el archivo original .csv a un .xlsx, pues me permite evitar algunos problemas de compatibilidad y me facilita tratar lo datos. Algo más que me gusta hacer es crear una copia del archivo original y trabajar con dicha copia. De esta manera puedo volver al archivo inicial en cualquier momento, en caso de que sea necesario.

Limpieza de los datos

Aquí se muestran algunas de las actividades que realicé como parte del proceso de limpieza de datos:

  • Quitar valores duplicados.
  • Añadir filtros y analizar los datos para encontrar cualquier error.
  • Cambiar el nombre de los encabezados, para hacerlos más fáciles de entender y más representativos de cada columna.
    • Antes Después
  • Reemplazar la información de algunas columnas como lo son "Sex" y "Chest Pain Type", para mostrar una versión más clara y detallada de cada celda.

  • Utilizar Ifs anidados, con la finalidad de separar las edades en varios grupos y optimizar su uso posterior en gráficas y visualizaciones.
    • Resultado
  • Ajustes de formato para hacer el contenido más agradable visualmente.

Tablas dinámicas

En esta parte creé y utilicé 3 tablas dinámicas. En la primera se muestra cómo varía la enfermedad cardíaca según el grupo de edad. La segunda tabla dinámica muestra el tipo de dolor de pecho de cada uno de los pacientes. En la tercera tabla se muestra cómo varía la enfermedad cardíaca de acuerdo al sexo del paciente.

Dashboard

Elaboré un "dashboard" con la finalidad de poder mostrar la información de una manera más agradable y organizada. Esto permite a los usuarios interpretar los datos, gráficas y estadísticas más fácilmente.

Agregué dos secciones de segmentación de datos, esto para permitir al usuario filtrar la información de manera más interactiva. En este caso, se utilizaron filtros para los valores de ECG en reposo (resting ECG) y niveles de glucosa en ayunas (fasting blood sugar results).

Por último, me dediqué un tiempo a mejorar la parte estética del dashboard, enfocándome en detalles como el diseño, la distribución y los colores. Esto ayuda a hacer el panel más agradable y atractivo para el usuario.
